# 电路模拟器 - 本地测试说明

## 方法1: 直接打开（推荐）

由于项目使用ES6模块，需要通过HTTP服务器运行。但可以尝试：

1. 右键点击 `index.html`
2. 选择"打开方式" → 选择您的浏览器
3. 如果浏览器阻止模块加载，请使用方法2

## 方法2: 使用VS Code Live Server

1. 在VS Code中安装"Live Server"扩展
2. 右键点击 `index.html`
3. 选择"Open with Live Server"
4. 浏览器会自动打开 http://127.0.0.1:5500

## 方法3: 使用Python（如果已安装）

在命令行中运行：
```powershell
# 查找Python
Get-Command python*

# 使用找到的命令启动服务器
python -m http.server 8080
# 或
py -m http.server 8080

# 然后在浏览器打开: http://localhost:8080
```

## 方法4: 使用Node.js（如果已安装）

```powershell
npx serve -l 8080
```
